For this assignment I made a calculator using HTML, CSS, and JavaScript. I then integrated it into my previously made personal portfolio. The calculator has a responsive layout with buttons and operators. It also has a screen that displays what is currently being typed. It has keyboard support so that computer users may type numbers and operators directly instead of using the buttons. I included some hover animations for the buttons to make it a little more unique. The design was inspired by both the iOS calculator and Android calculator design, using like colors and layout. The calculator has been integrated into my portfolio and is accessible from the projects tab.
